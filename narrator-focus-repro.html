Narrator Repro: Combobox Triggers Scan Mode

Steps to reproduce:
  1. Enable Narrator (Win + Ctrl + Enter)
  2. Tab to the editor and type any character
  3. The listbox appears with the first item pre-selected

Expected: Narrator stays in forms mode and reads the active option via aria-activedescendant.

Actual: Narrator flips into scan mode.

Root cause: aria-activedescendant is set on the combobox at the same moment the listbox is dynamically inserted into the DOM.
